# Server Side

## Installation
```
npm install
cp .env.example .env
```

## Run

### Compile and Hot-reload for Development
```
npm run dev
```

Open [http://localhost:4000](http://localhost:4000) to view it in the browser.

### Compile and Minify for Production
```
npm run build
```

### Start the Production Server
```
npm run start
```
